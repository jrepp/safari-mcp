# LinkedIn (עברית) — safari-mcp

## טקסט הפוסט

```
ה-AI agent שלכם מריץ עכשיו שני דפדפנים בו-זמנית.

Safari — עם כל ההתחברויות שלכם. GitHub. Gmail. בנק. תוספי
סיסמאות. 2FA. זה שאתם באמת משתמשים בו.

ו-Chromium headless ש-Playwright הדליק. בלי cookies. בלי
סשנים. בלי היסטוריה. זר מוחלט.

כשה-agent צריך "ללכת לבדוק משהו" — הוא נוהג בזר. נתקל
בדף התחברות. מבקש מכם סיסמה. נופל בכל מקרה כי יש 2FA.

שנה שלמה התעצבנתי מזה. אז הוצאתי Safari MCP — שרת MCP
שנוהג ב-Safari שאתם כבר מחוברים בו. 80 כלים. בלי Chromium.
בלי Puppeteer. בלי לופים של התחברות מחדש.

~60% פחות CPU מ-Chrome headless על Apple Silicon (מדידה על
M1 Pro עם אותה לולאת scraping).

6,000+ הורדות חודשיות ב-npm תוך 30 יום ראשונים.

אם אתם בונים agents על Mac, ויש לכם Safari פתוח כבר — אתם
מריצים דפדפן שני בלי סיבה.

לינק בתגובה הראשונה.
```

## תגובה ראשונה (מיד אחרי הפרסום)
